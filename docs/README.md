## Architecture Decision Records (ADRs)

ADRs document significant technical decisions. Create one whenever you make a choice that:
- Affects multiple team members
- Is hard or expensive to reverse
- Involves trade-offs worth recording

### Format
```markdown
# ADR-001: [Title]

## Status: Accepted / Proposed / Deprecated

## Context
What is the problem or situation?

## Decision
What did we decide?

## Consequences
What are the trade-offs? What becomes easier/harder?
```
